# poly2geojson

Convert Osmosis polygon filter file to GeoJSON polygon.

## Install

```sh
$ npm install --save poly2geojson
```

## Usage

```js
var poly2geojson = require('poly2geojson');

poly2geojson('Rainbow');
```
